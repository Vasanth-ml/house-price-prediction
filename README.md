# House Price Prediction

## Problem Statement
Predict house prices based on features like 
area, location, number of rooms and other 
property characteristics using Machine Learning.

## Dataset
- Source: Kaggle - Ames Housing Dataset
- Size: 1460 rows, 81 features
- Target: Sale Price (house price in USD)

## Algorithm Used
- Linear Regression
- Random Forest Regressor

## Results
- Linear Regression R² Score: 0.89
- Random Forest R² Score: improved performance

## Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Steps Performed
1. Data Loading and Exploration
2. Data Cleaning (missing values)
3. Feature Engineering
4. Train Test Split (80/20)
5. Model Training
6. Model Evaluation (MAE, MSE, R²)
7. Visualization

## Key Learnings
- How to handle missing values in real datasets
- Feature importance in house price prediction
- Comparing multiple regression models
